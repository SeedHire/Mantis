Why Mantis

Built for developers who want serious tooling without the $30/month AI tax.

🧠

Persistent Project Memory

Mantis writes a .mantis/BRAIN.md after every session. It remembers your architecture decisions, rejected approaches, and patterns — forever.

⚡

Codebase Intelligence

Run mantis init once to build a live AST dependency graph. Every question gets only the relevant files injected — not your whole repo.

🎯

Smart Model Routing

Every message is classified — trivial, fast, code, reasoning, or heavy — and routed to the right model automatically. No wasted tokens on simple questions.

🔍

Hallucination Guard

AI output is checked against your actual function signatures in real time. If the model invents a function name, Mantis flags it before you act on it.

📐

Architecture Lint

Define rules like "no controller may import from db layer" in .mantisrc.yml and enforce them in CI — just like ESLint but for your architecture.

💰

Token Cost Tracker

At the end of every session, see exactly what you would have paid with GPT-4o, Claude Sonnet, or Claude Opus — vs the $0.00 you spent with Mantis.

🖼️

Multimodal Input

Paste a screenshot, architecture diagram, or error image with /vision or the --image flag. Works with any vision-capable Ollama model.

🔌

Fully Offline

No data leaves your machine. Your code, your history, your memory — all local. Use Ollama Cloud for convenience or your own GPU for absolute privacy.

📊

TUI Dashboard

Run mantis tui for a full terminal dashboard — symbol search, impact analysis, dead code, and architecture lint in one place.

Getting started

Up and running in 3 steps.

No account. No API key. Just install and go.

1

Install Mantis

One curl command installs the binary for macOS, Linux, or Windows.


curl -fsSL …/install.sh | sh
2

Start Ollama

Install Ollama locally or get a free cloud key at ollama.com/cloud. No credit card needed.


ollama serve
3

Index your project

Run mantis init once in your project root to build the AST graph. Then just type mantis.


mantis init && mantis
Slash commands

Powerful commands, simple interface.

Everything you need is one slash away.

/file <path>

Inject a file into the conversation context — precise and token-efficient.

/vision <img>

Analyze a screenshot, error dialog, or architecture diagram.

/brain

View the accumulated project memory Mantis has built up over your sessions.

/save

Force-save the current session to project memory right now.

/cost

Show the session token report — what you used vs what it would have cost elsewhere.

/reset

Clear conversation history while keeping all brain memory intact.

/decision

Log an architecture decision. Mantis will reference it in future sessions.

/reject

Mark the last suggestion as a rejected approach so Mantis never repeats it.

Project Memory

It remembers. Across every session.

Mantis writes plain text files you can read, edit, and commit. No proprietary format, no vendor lock-in.

  • ✓BRAIN.md — rolling project summary, updated each session
  • ✓DECISIONS.log — timestamped architecture decisions
  • ✓REJECTED.md — approaches the AI tried and you rejected
  • ✓GROUND_TRUTH.json — live function signatures for hallucination checking
