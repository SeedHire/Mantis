Get started

Up and running in 3 steps.

1

Install

Run brew install seedhire/tap/mantis or curl -fsSL .../install.sh | sh. One binary, no dependencies.

2

Sign in with GitHub

Mantis opens a browser tab on first run. One click to authorize — no extra account, no billing setup.

3

Index + start

Run mantis init once, then use mantis (or --image) to start with full graph-aware context.

Project Memory

Persistent memory with verification built in.

Mantis stores human-readable project memory, session context, semantic embeddings, and a live ground-truth signature index. On startup, it reloads this state so answers stay consistent with your actual codebase and previous decisions.

.mantis/
  BRAIN.md  ← rolling project summary
  CONVENTIONS.md  ← architecture rules
  DECISIONS.log  ← timestamped decisions
  REJECTED.md  ← approaches that failed
  GROUND_TRUTH.json  ← real function signatures
  embeddings.db  ← semantic memory (BM25 + cosine hybrid)
  graph.db  ← AST dependency graph
  AGENT_SCRATCH.json  ← multi-agent communication
  last-pipeline.md  ← full pipeline output
  sessions/  ← persisted chat history
  skills/  ← built-in + custom skill files
.mantisrc.yml  ← CI lint rules
